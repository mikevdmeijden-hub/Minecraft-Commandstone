Commandstone is a searchable reference and command builder for Minecraft: Java Edition. Everything runs offline in this window.

Search
Type in the search box to filter across every command, entity attribute, and mob at once. Use the All / Commands / Attributes / Mobs chips to narrow what's listed. The result count updates as you type.

Build a command
Click any item in the list and it opens on the right:
  •  Commands open a builder — fill in the fields and the finished command updates live in the preview. Press Copy command to send it to your clipboard.
  •  Attributes become a ready /attribute … base set … command.
  •  Mobs become a /summon … command with editable coordinates.
Commands without a dedicated builder show an editable syntax line you can tweak and copy.

Themes
Use the toggle in the top-right corner, or the View menu, to switch between dark and light. Your current selection is kept when you switch.

Good to know
Data reflects current Java Edition (26.x). Attribute IDs are flattened (for example minecraft:max_health) and gamerules use snake_case names. Minecraft changes quickly — in-game tab-completion is always the final word for your exact version.
